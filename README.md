Loja de Roupa - E-commerce Front-end

Projeto de uma vitrine de moda minimalista com sistema de carrinho dinâmico e integração híbrida para finalização de pedidos.

Tecnologias Utilizadas
HTML5: Estruturação semântica.
CSS3: Layout responsivo com Flexbox, Grid e efeitos de Backdrop-filter (Glassmorphism).
JavaScript (ES6+): Manipulação de DOM, gerenciamento de estado do carrinho e cálculos de soma total.

Funcionalidades
- [x] Adição de produtos ao carrinho sem recarregar a página.
- [x] Cálculo automático de subtotal e total.
- [x] Carrinho lateral interativo (Sidebar).
- [x] Checkout direto via API do WhatsApp.
- [x] Design totalmente responsivo (Desktop e Mobile).
